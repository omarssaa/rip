Matter consists of atoms and atoms have different types, each type is an element.

There is currently 118 different elements that have been discovered. Obviously, memorising this big number of elements is a tedious work. So, we need something that will contain at least basic information about all the elements. Here, the periodic table steps in.

The periodic table is a table that contains basic information about all the elements, like their atomic number, symbol, atomic mass and the number of valence electrons.

As we can see from the table above, the periodic table has 18 vertical groups and 7 horizontal periods. Each group contains elements with similar properties and electronic configuration, while the elements in a period have the same number of electron shells.

Periodic properties of the elements:

  • The metallic property decreases as we go from left to right through the same period, while it increases from top to down through the same group.
  • The atomic radius decreases from left to right through the same period, while it increases from top to bottom through the same group.
  • The ionization energy increases from left to right through the same period, while it decreases from top to bottom through the same group.

Elements can be classified into 3 main groups: Metals, Metalloids(semi-metals) and Non-metals.

Metals usually form ionic compounds with non-metals, metallic bonds with metals, and in some cases metals can form covalent bonds with non-metals and/or metals.


On the left is dilithium, a metal-metal covalent bond. In the middle, sodium oxide, an example of ionic bond. And on the right Aluminum chloride, an example of metal to non-metal covalent bond.
